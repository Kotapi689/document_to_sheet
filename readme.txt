※インストール所要時間は3〜5分です。

【できること】
ドキュメントに原稿が書けたら、依頼台本スプシを開いてマクロをポチっとするだけで
・原稿をスプシに転記
・話者の振り分け（霊夢 / 魔理沙 / 霊夢&魔理沙）
・崩れた部分を整形する
この3点を1分かからずやってくれます。


【インストール手順】
最初だけ必要な作業を箇条書きします。
1.sheetscript.txtの中身を開いて、全文コピーする
2.依頼台本スプシを開いて「拡張機能」→「Apps Script」をクリック
3.出てきたエディターにそのままペースト
4.ペーストしたスクリプトの上10行くらいの設定項目を埋めて保存（⌘+S / Ctrl+S）
5.エディタ画面の左の方に歯車マーク「プロジェクトの設定」があるのでそれを押す
6.「appsscript.json」マニフェストファイルをエディタで表示する→チェック入れる
7.歯車の上、< >←こんなマークの「エディタ」を押して戻るとappsscript.jsonが増えてるはず。僕が作ったappsscript.json.txtファイルを開いて、中身をまるごとコピー→ペーストする(元ある記述は消してからペースト)
8.エディタはもう閉じてOK！スプシに戻って「拡張機能」→「マクロ」→「マクロをインポート」→autoFillを見つけたら「関数を追加」
※エディタで編集するとその都度マクロのインポートが必要です

ここまでできれば最初だけの作業はほぼ終了！


【初期設定補足】
上4つは
・セリフ入力欄がどのセルから始まるか？
・話者の入力欄がどのセルから始まるか？
・文字数カウントがどのセルから始まるか？
この3点をスクリプトに教えてあげるための設定です。
行=1,2,3,4,.... 横の数字です
列=A,B,C,D,.... 縦のアルファベットです

下2つは、霊夢&魔理沙の2人で発話するところでいつも何言わせてるか？を教えてあげる項目です。
キーワードを含むかどうかで判定するので、句読点あるかないか、とか細かい違いでスルーすることもありません。


【使い方】
インストールも終わっていて、ドキュメントで原稿が書けた体で説明します。
1.ドキュメントのIDをコピーしておく（アドレス欄の /d/xxxxxxxxxxxxxx/edit ←この xxxx部分がID）
2.依頼台本スプシを開いて「拡張機能」→「マクロ」→「autoFill」を押すとスクリプト発動
3.3つ質問するので答える。1でコピーしたドキュメントIDもここで使う。→選択肢「はい」がなぜか「OK」になるけど問題なく動きます
4.「完了しました！」のポップアップが出たら終了


【「承認が必要です」エラーの対処法】
始めてスクリプトを実行する時は権限に関するアラートが出ます。
1.「権限を確認」
2.Googleにログインする
3.ちっちゃい「詳細」リンクを押す
4.「〇〇のプロジェクト（安全ではないページ）に移動」を押す
5.「許可」を押す
この5ステップでクリアできます。二度と聞かれません。


【このマクロがハマる原稿の書き方】
話者が切り替わるタイミングで1行開ける。それ以外は行間を開けない。これさえ気をつければ大体OKです。
